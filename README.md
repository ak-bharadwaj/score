# score

Quick Deploy (Render)
- Click the button below to deploy the app (frontend + backend + WebSockets) as one Docker service on Render. Provide your MongoDB Atlas connection string and Render will generate secrets.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/ak-bharadwaj/score)

Environment Variables
- MONGO_URI: MongoDB Atlas connection string
- ACCESS_TOKEN_SECRET: generated by Render (or set your own)
- REFRESH_TOKEN_SECRET: generated by Render (or set your own)
- PORT: 5000 (default)
- CORS_ORIGINS: optional, comma-separated additional origins

Local Development
```bash
docker compose up -d --build
# App: http://localhost:5000
```
